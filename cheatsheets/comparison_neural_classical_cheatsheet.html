üî∑ 4. Transfer Learning

  • Discriminative learning rates: —Ä–∞–∑–Ω—ã–µ LR –¥–ª—è —Å–ª–æ—ë–≤
  • Gradual unfreezing: –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–æ–µ —Ä–∞–∑–º–æ—Ä–∞–∂–∏–≤–∞–Ω–∏–µ
  • Fine-tuning: –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∞—è –Ω–∞—Å—Ç—Ä–æ–π–∫–∞

üî∑ 8. Learning Rate Policies

1cycle Policy:

  • Warm-up: –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–æ–µ —É–≤–µ–ª–∏—á–µ–Ω–∏–µ LR
  • Annealing: –ø–æ—Å—Ç–µ–ø–µ–Ω–Ω–æ–µ —É–º–µ–Ω—å—à–µ–Ω–∏–µ
  • Momentum: –æ–±—Ä–∞—Ç–Ω–æ –ø—Ä–æ–ø–æ—Ä—Ü–∏–æ–Ω–∞–ª—å–Ω–æ LR
# 1cycle training
learn.fit_one_cycle(
    n_epoch=10,
    lr_max=1e-3,
    moms=(0.95, 0.85),  # momentum range
    div=25.0,            # initial LR division
    pct_start=0.3        # warmup percentage
)

# Flat + cosine annealing
learn.fit_flat_cos(
    n_epoch=10,
    lr=1e-3,
    pct_start=0.75
)
