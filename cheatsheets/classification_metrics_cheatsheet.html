üî∑ 1. –û—Å–Ω–æ–≤–Ω—ã–µ –ø–æ–Ω—è—Ç–∏—è

  • TP (True Positive): –ø—Ä–∞–≤–∏–ª—å–Ω–æ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π –∫–ª–∞—Å—Å
  • TN (True Negative): –ø—Ä–∞–≤–∏–ª—å–Ω–æ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π –∫–ª–∞—Å—Å
  • FP (False Positive): –ª–æ–∂–Ω–∞—è —Ç—Ä–µ–≤–æ–≥–∞ (Type I error)
  • FN (False Negative): –ø—Ä–æ–ø—É—Å–∫ (Type II error)

üî∑ 2. Confusion Matrix

üî∑ 3. Accuracy (–¢–æ—á–Ω–æ—Å—Ç—å)

–§–æ—Ä–º—É–ª–∞: (TP + TN) / (TP + TN + FP + FN)

from sklearn.metrics import accuracy_score

accuracy = accuracy_score(y_test, y_pred)
print(f"Accuracy: {accuracy:.3f}")

–ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:

  • ‚úÖ –°–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∫–ª–∞—Å—Å—ã
  • ‚ùå –ù–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∫–ª–∞—Å—Å—ã (misleading!)

üî∑ 4. Precision (–¢–æ—á–Ω–æ—Å—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω–æ–≥–æ –∫–ª–∞—Å—Å–∞)

–§–æ—Ä–º—É–ª–∞: TP / (TP + FP)

–í–æ–ø—Ä–æ—Å: "–ò–∑ –≤—Å–µ—Ö –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω—ã—Ö –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö, —Å–∫–æ–ª—å–∫–æ –¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö?"

from sklearn.metrics import precision_score

precision = precision_score(y_test, y_pred)
print(f"Precision: {precision:.3f}")

–ö–æ–≥–¥–∞ –≤–∞–∂–Ω–∞:

  • –°—Ç–æ–∏–º–æ—Å—Ç—å FP –≤—ã—Å–æ–∫–∞ (—Å–ø–∞–º-—Ñ–∏–ª—å—Ç—Ä)
  • –ú–µ–¥–∏—Ü–∏–Ω—Å–∫–∞—è –¥–∏–∞–≥–Ω–æ—Å—Ç–∏–∫–∞ (–Ω–µ –ø—É–≥–∞—Ç—å –∑–¥–æ—Ä–æ–≤—ã—Ö)

üî∑ 5. Recall (–ü–æ–ª–Ω–æ—Ç–∞, Sensitivity)

–§–æ—Ä–º—É–ª–∞: TP / (TP + FN)

–í–æ–ø—Ä–æ—Å: "–ò–∑ –≤—Å–µ—Ö —Ä–µ–∞–ª—å–Ω—ã—Ö –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö, —Å–∫–æ–ª—å–∫–æ –º—ã –Ω–∞—à–ª–∏?"

from sklearn.metrics import recall_score

recall = recall_score(y_test, y_pred)
print(f"Recall: {recall:.3f}")

–ö–æ–≥–¥–∞ –≤–∞–∂–µ–Ω:

  • –°—Ç–æ–∏–º–æ—Å—Ç—å FN –≤—ã—Å–æ–∫–∞ (–ø—Ä–æ–ø—É—Å–∫ –±–æ–ª–µ–∑–Ω–∏)
  • Fraud detection (–Ω–∞–π—Ç–∏ –≤—Å–µ –º–æ—à–µ–Ω–Ω–∏—á–µ—Å—Ç–≤–∞)

üî∑ 6. F1-Score

–§–æ—Ä–º—É–ª–∞: 2 * (Precision * Recall) / (Precision + Recall)

–ì–∞—Ä–º–æ–Ω–∏—á–µ—Å–∫–æ–µ —Å—Ä–µ–¥–Ω–µ–µ precision –∏ recall

from sklearn.metrics import f1_score

f1 = f1_score(y_test, y_pred)
print(f"F1 Score: {f1:.3f}")

–ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å:

  • –ù–µ—Å–±–∞–ª–∞–Ω—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∫–ª–∞—Å—Å—ã
  • –ù—É–∂–µ–Ω –±–∞–ª–∞–Ω—Å precision/recall
  • –û–±—â–∞—è –º–µ—Ç—Ä–∏–∫–∞ –∫–∞—á–µ—Å—Ç–≤–∞

üî∑ 9. ROC-AUC

ROC: Receiver Operating Characteristic

AUC: Area Under Curve (0.5-1.0)

from sklearn.metrics import roc_auc_score, roc_curve

# –ù—É–∂–Ω—ã –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏, –∞ –Ω–µ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏—è

auc = roc_auc_score(y_test, y_proba)
print(f"ROC-AUC: {auc:.3f}")

# –ü–æ—Å—Ç—Ä–æ–µ–Ω–∏–µ ROC –∫—Ä–∏–≤–æ–π
fpr, tpr, thresholds = roc_curve(y_test, y_proba)

plt.figure(figsize=(8, 6))
plt.plot(fpr, tpr, label=f'ROC curve (AUC = {auc:.2f})')
plt.plot([0, 1], [0, 1], 'k--', label='Random')
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
plt.title('ROC Curve')
plt.legend()
plt.show()
