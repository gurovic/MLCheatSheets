üî∑ 2. –°—Ç—Ä—É–∫—Ç—É—Ä–∞ (–±–∏–Ω–∞—Ä–Ω–∞—è)

Actual: PositiveTP (True Positive)FN (False Negative)
Actual: NegativeFP (False Positive)TN (True Negative)

–†–∞—Å—à–∏—Ñ—Ä–æ–≤–∫–∞:

  • TP: –ø—Ä–∞–≤–∏–ª—å–Ω–æ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–ª–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π –∫–ª–∞—Å—Å
  • TN: –ø—Ä–∞–≤–∏–ª—å–Ω–æ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–ª–∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π –∫–ª–∞—Å—Å
  • FP: –æ—à–∏–±–æ—á–Ω–æ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–ª–∏ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–π (Type I error)
  • FN: –æ—à–∏–±–æ—á–Ω–æ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–ª–∏ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π (Type II error)

from sklearn.metrics import confusion_matrix
import numpy as np

# –†–µ–∞–ª—å–Ω—ã–µ –∏ –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è
y_true = [0, 1, 0, 1, 0, 1, 1, 0]
y_pred = [0, 1, 0, 0, 0, 1, 1, 1]

# Confusion matrix

print(cm)
# [[3 1]
#  [1 3]]

# –ò–∑–≤–ª–µ—á–µ–Ω–∏–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç–æ–≤

print(f"TP: {tp}, TN: {tn}, FP: {fp}, FN: {fn}")

üî∑ 5. –ú–µ—Ç—Ä–∏–∫–∏ –∏–∑ –º–∞—Ç—Ä–∏—Ü—ã

–ú–µ—Ç—Ä–∏–∫–∞–§–æ—Ä–º—É–ª–∞–ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è
Accuracy(TP+TN)/(TP+TN+FP+FN)–û–±—â–∞—è —Ç–æ—á–Ω–æ—Å—Ç—å
PrecisionTP/(TP+FP)–¢–æ—á–Ω–æ—Å—Ç—å –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã—Ö
Recall (Sensitivity)TP/(TP+FN)–ü–æ–ª–Ω–æ—Ç–∞
SpecificityTN/(TN+FP)–ü–æ–ª–Ω–æ—Ç–∞ –æ—Ç—Ä–∏—Ü–∞—Ç–µ–ª—å–Ω—ã—Ö
F1-Score2*(Precision*Recall)/(Precision+Recall)–ì–∞—Ä–º–æ–Ω–∏—á–µ—Å–∫–æ–µ —Å—Ä–µ–¥–Ω–µ–µ

üî∑ 6. –í—ã—á–∏—Å–ª–µ–Ω–∏–µ –º–µ—Ç—Ä–∏–∫

from sklearn.metrics import (
    accuracy_score, precision_score, 
    recall_score, f1_score
)

# –ò–∑ confusion matrix
tn, fp, fn, tp = confusion_matrix(y_true, y_pred).ravel()

accuracy = (tp + tn) / (tp + tn + fp + fn)
precision = tp / (tp + fp)
recall = tp / (tp + fn)
specificity = tn / (tn + fp)
f1 = 2 * (precision * recall) / (precision + recall)

print(f"Accuracy: {accuracy:.3f}")
print(f"Precision: {precision:.3f}")
print(f"Recall: {recall:.3f}")
print(f"Specificity: {specificity:.3f}")
print(f"F1-Score: {f1:.3f}")

# –ò–ª–∏ –Ω–∞–ø—Ä—è–º—É—é –∏–∑ sklearn
print(f"Accuracy: {accuracy_score(y_true, y_pred):.3f}")
print(f"Precision: {precision_score(y_true, y_pred):.3f}")
print(f"Recall: {recall_score(y_true, y_pred):.3f}")
print(f"F1-Score: {f1_score(y_true, y_pred):.3f}")
