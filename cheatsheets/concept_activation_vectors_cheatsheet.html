üî∑ 3. –ú–∞—Ç–µ–º–∞—Ç–∏–∫–∞ TCAV

CAV –≤—ã—á–∏—Å–ª–µ–Ω–∏–µ:

–û–±—É—á–∏—Ç—å –ª–∏–Ω–µ–π–Ω—ã–π –∫–ª–∞—Å—Å–∏—Ñ–∏–∫–∞—Ç–æ—Ä: w¬∑h + b = 0

CAV = w/||w|| (–Ω–æ—Ä–º–∞–ª–∏–∑–æ–≤–∞–Ω–Ω—ã–π –≤–µ–∫—Ç–æ—Ä –≤–µ—Å–æ–≤)

Directional derivative:

SC,k,l(x) = ‚àáhlfk(x) ¬∑ vCl

–≥–¥–µ:

  • fk - –ø—Ä–µ–¥—Å–∫–∞–∑–∞–Ω–∏–µ –∫–ª–∞—Å—Å–∞ k
  • hl - –∞–∫—Ç–∏–≤–∞—Ü–∏–∏ —Å–ª–æ—è l
  • vCl - CAV –¥–ª—è –∫–æ–Ω—Ü–µ–ø—Ü–∏–∏ C –≤ —Å–ª–æ–µ l

TCAV score:

TCAVC,k,l = |{x ‚àà Xk : SC,k,l(x) > 0}| / |Xk|

def get_activations(model, images, layer_name):
    activations = []
    def hook(module, input, output):
        activations.append(output.detach())
    
    handle = getattr(model, layer_name).register_forward_hook(hook)
    model(images)
    handle.remove()
    return activations[0]

def compute_tcav_score(model, test_images, cav, layer_name):
    tcav_scores = []
    
    for img in test_images:
        img.requires_grad = True
        
        # Forward pass
        activations = get_activations(model, img.unsqueeze(0), layer_name)
        output = model(img.unsqueeze(0))
        
        # Gradient
        output[0, target_class].backward()
        grad = activations.grad
        
        # Directional derivative
        directional_deriv = torch.sum(grad * torch.tensor(cav))
        tcav_scores.append(directional_deriv.item() > 0)
    
    return sum(tcav_scores) / len(tcav_scores)
