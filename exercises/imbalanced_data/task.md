# Упражнение: Балансировка классов (Imbalanced Data)

## Задание

Дан файл `input.csv` с несбалансированными данными для бинарной классификации. Необходимо:

1. Определить количество образцов в каждом классе (class 0 и class 1)

2. Применить **Random Oversampling** для меньшинства класса (class 1):
   - Случайно дублировать образцы из класса 1 до достижения баланса
   - Использовать random_state=42 для воспроизводимости

3. После балансировки класс 1 должен иметь столько же образцов, сколько класс 0

4. Сохранить результат: отсортировать по id, затем по всем признакам

## Метод Random Oversampling

1. Посчитать количество образцов в большинстве классе (class 0)
2. Случайным образом выбрать образцы из меньшинства класса (class 1) с повторением
3. Добавить дублированные образцы к исходным данным
4. Использовать random_state=42 для np.random.seed()

## Ожидаемый формат выходного файла

CSV файл с колонками: id, feature1, feature2, feature3, class

Новые дублированные строки должны иметь новые id, начиная с максимального id + 1.
Результат должен быть отсортирован по id.
