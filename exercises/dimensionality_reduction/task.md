# Упражнение: Снижение размерности (Dimensionality Reduction)

## Задание

Дан файл `input.csv` с 4-мерными данными. Необходимо:

1. Применить **PCA (Principal Component Analysis)** для снижения размерности до 2 компонент

2. Использовать StandardScaler перед применением PCA (центрирование и масштабирование)

3. Сохранить результат: id и две главные компоненты (PC1, PC2)

## Алгоритм PCA

1. Стандартизировать данные: z = (x - μ) / σ
2. Вычислить ковариационную матрицу
3. Найти собственные векторы и собственные значения
4. Выбрать топ-2 собственных вектора (с наибольшими собственными значениями)
5. Спроецировать данные на эти векторы

## Примечание

Используйте библиотеку sklearn для реализации:
- StandardScaler для стандартизации
- PCA(n_components=2) для снижения размерности

Округлите результаты до 4 знаков после запятой.

## Ожидаемый формат выходного файла

CSV файл с колонками: id, PC1, PC2

## Ответ

Результат должен находиться в файле `answer.csv`
