# Упражнение: Обработка пропущенных значений и кодирование

## Задание

Дан файл `input.csv` с данными о сотрудниках, содержащий пропущенные значения. Необходимо:

1. Удалить колонку 'name' (она не используется для анализа)

2. Заполнить пропущенные значения:
   - Для числовых колонок (age, salary, experience) использовать **среднее значение** (mean) существующих значений
   - Для категориальной колонки (city) использовать **моду** (наиболее частое значение)

3. Применить One-Hot Encoding к колонке 'city'
   - Создать бинарные колонки для каждого уникального города
   - Порядок колонок: Moscow, SPB (алфавитный порядок)

4. Сохранить результат с колонками: age, salary, experience, city_Moscow, city_SPB

## Правила заполнения пропусков

- Mean округляется до целых для age и experience, до 2 знаков после запятой для salary
- Мода для city: выбирается наиболее частое значение

## Правила One-Hot Encoding

- city_Moscow: 1 если город Moscow, иначе 0
- city_SPB: 1 если город SPB, иначе 0

## Ожидаемый формат выходного файла

CSV файл с колонками: age, salary, experience, city_Moscow, city_SPB

## Ответ

Результат должен находиться в файле `answer.csv`
