# Упражнение 3: Рекурсивное удаление признаков (RFE)

## Задание

Дан файл `input.csv` с признаками и целевой переменной для задачи классификации. Необходимо:

1. Применить **RFE (Recursive Feature Elimination)** с Random Forest для выбора **топ-3 признаков**

2. Сохранить результат в виде названий выбранных признаков в алфавитном порядке

## Что такое RFE?

RFE (Recursive Feature Elimination) - это метод выбора признаков, который:
1. Обучает модель на всех признаках
2. Удаляет наименее важный признак (по важности модели)
3. Повторяет процесс, пока не останется нужное количество признаков

## Алгоритм

1. Использовать `sklearn.feature_selection.RFE` с оценщиком `RandomForestClassifier`
2. Параметры:
   - `n_features_to_select=3` - выбрать 3 признака
   - `random_state=42` для RandomForestClassifier
   - `n_estimators=100` для RandomForestClassifier
3. Отсортировать выбранные признаки в алфавитном порядке

## Ожидаемый формат выходного файла

CSV файл с одной колонкой 'feature', содержащей названия выбранных признаков:
```
feature
feature1
feature2
feature5
```

## Примечание

Для получения однозначного результата используйте указанные параметры.
