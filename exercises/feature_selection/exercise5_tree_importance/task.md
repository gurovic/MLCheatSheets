# Упражнение 5: Выбор признаков по важности дерева (Tree-based Importance)

## Задание

Дан файл `input.csv` с признаками и целевой переменной для задачи классификации. Необходимо:

1. Обучить **Random Forest** и получить важность признаков (feature importance)

2. Выбрать **топ-4 признака** с наибольшей важностью

3. Сохранить результат в виде названий выбранных признаков в порядке убывания важности

## Что такое Feature Importance?

Feature Importance (важность признаков) в древовидных моделях показывает, насколько каждый признак полезен для построения дерева решений. Вычисляется на основе среднего уменьшения Gini impurity.

## Алгоритм

1. Использовать `sklearn.ensemble.RandomForestClassifier` с параметрами:
   - `n_estimators=100`
   - `random_state=42`
2. Обучить модель и получить `feature_importances_`
3. Выбрать топ-4 признака с максимальной важностью
4. Отсортировать по убыванию важности

## Ожидаемый формат выходного файла

CSV файл с одной колонкой 'feature', содержащей названия выбранных признаков:
```
feature
feature3
feature1
feature5
feature2
```

## Примечание

Для получения однозначного результата используйте `random_state=42`.
