# Упражнение 6: Удаление коррелирующих признаков

## Задание

Дан файл `input.csv` с признаками. Необходимо:

1. Вычислить **корреляционную матрицу** признаков (feature1-feature6)

2. Удалить признаки, которые имеют **корреляцию > 0.8** с другими признаками

3. Сохранить результат в виде названий оставшихся признаков в алфавитном порядке

## Зачем удалять коррелирующие признаки?

Высококоррелирующие признаки:
- Содержат избыточную информацию
- Могут вызывать мультиколлинеарность
- Замедляют обучение модели
- Усложняют интерпретацию

## Алгоритм

1. Вычислить матрицу корреляции Пирсона между всеми признаками
2. Рассмотреть только верхний треугольник матрицы (без диагонали)
3. Для каждой пары признаков с |корреляцией| > 0.8:
   - Удалить признак, который идет позже по алфавиту (второй в паре)
   - Например: если feature1 коррелирует с feature3, удаляем feature3
4. Отсортировать оставшиеся признаки в алфавитном порядке

## Пример

Если feature1 и feature3 имеют корреляцию 0.85:
- Удаляем feature3 (т.к. она позже по алфавиту)

## Ожидаемый формат выходного файла

CSV файл с одной колонкой 'feature', содержащей названия оставшихся признаков:
```
feature
feature1
feature2
feature5
```

## Ответ

Результат должен находиться в файле `answer.csv`
