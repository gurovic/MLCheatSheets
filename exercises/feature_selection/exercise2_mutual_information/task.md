# Упражнение 2: Выбор признаков с помощью Mutual Information

## Задание

Дан файл `input.csv` с признаками и целевой переменной для задачи классификации. Необходимо:

1. Вычислить **Mutual Information** между каждым признаком (feature1-feature5) и целевой переменной (target)

2. Выбрать **топ-3 признака** с наибольшими значениями MI

3. Сохранить результат в виде названий выбранных признаков в порядке убывания MI

## Что такое Mutual Information?

Mutual Information (взаимная информация) измеряет, насколько знание одной переменной уменьшает неопределенность другой. 
Значения MI всегда неотрицательные: MI = 0 означает независимость переменных.

## Алгоритм

1. Использовать `sklearn.feature_selection.mutual_info_classif` с параметром `random_state=42`
2. Отобрать топ-3 признака с максимальными значениями MI
3. Отсортировать по убыванию MI

## Ожидаемый формат выходного файла

CSV файл с одной колонкой 'feature', содержащей названия выбранных признаков:
```
feature
feature3
feature1
feature5
```

## Примечание

Для получения однозначного результата используйте `random_state=42` при вычислении mutual information.
