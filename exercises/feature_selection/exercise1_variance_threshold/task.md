# Упражнение 1: Удаление низкодисперсных признаков (Variance Threshold)

## Задание

Дан файл `input.csv` с набором признаков. Необходимо:

1. Вычислить **дисперсию** для каждого признака (feature1-feature6)

2. Удалить признаки с дисперсией **меньше 0.5**

3. Сохранить результат в виде названий оставшихся признаков в алфавитном порядке

## Формула дисперсии

σ² = Σ(x - x̄)² / n

где:
- x - значения признака
- x̄ - среднее значение
- n - количество наблюдений

## Алгоритм

1. Вычислить дисперсию каждого признака
2. Отобрать признаки с дисперсией >= 0.5
3. Отсортировать названия признаков в алфавитном порядке

## Ожидаемый формат выходного файла

CSV файл с одной колонкой 'feature', содержащей названия оставшихся признаков:
```
feature
feature1
feature3
feature5
```

## Ответ

Результат должен находиться в файле `answer.csv`
