# Упражнение: Выбор признаков (Feature Selection)

## Задание

Дан файл `input.csv` с данными для бинарной классификации. Необходимо:

1. Вычислить **корреляцию Пирсона** между каждым признаком (feature1-feature5) и целевой переменной (target)

2. Выбрать **топ-3 признака** с наибольшей абсолютной корреляцией с target

3. Сохранить результат в виде названий выбранных признаков в порядке убывания абсолютной корреляции

## Формула корреляции Пирсона

r = Σ((x - x̄)(y - ȳ)) / √(Σ(x - x̄)² × Σ(y - ȳ)²)

где:
- x - значения признака
- y - значения target
- x̄, ȳ - средние значения

## Алгоритм

1. Вычислить корреляцию каждого признака с target
2. Взять абсолютное значение корреляций
3. Выбрать топ-3 признака с максимальной абсолютной корреляцией
4. Отсортировать по убыванию абсолютной корреляции

## Ожидаемый формат выходного файла

CSV файл с одной колонкой 'feature', содержащей названия выбранных признаков:
```
feature
feature2
feature1
feature4
```
