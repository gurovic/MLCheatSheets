# Упражнение 4: Выбор признаков с помощью Lasso (L1 регуляризация)

## Задание

Дан файл `input.csv` с признаками и целевой переменной для задачи регрессии. Необходимо:

1. Применить **Lasso регрессию** (L1 регуляризация) для выбора признаков

2. Выбрать признаки с **ненулевыми коэффициентами**

3. Сохранить результат в виде названий выбранных признаков в алфавитном порядке

## Что такое Lasso?

Lasso (Least Absolute Shrinkage and Selection Operator) - это метод регрессии с L1 регуляризацией, который:
- Автоматически выполняет выбор признаков
- Устанавливает коэффициенты незначимых признаков точно в 0
- Помогает создавать разреженные модели

## Алгоритм

1. Стандартизировать признаки с помощью `StandardScaler`
2. Использовать `sklearn.linear_model.Lasso` с параметрами:
   - `alpha=0.1` - коэффициент регуляризации
   - `random_state=42`
3. Отобрать признаки с коэффициентами != 0
4. Отсортировать в алфавитном порядке

## Ожидаемый формат выходного файла

CSV файл с одной колонкой 'feature', содержащей названия выбранных признаков:
```
feature
feature1
feature3
feature4
```

## Примечание

Обязательно стандартизируйте данные перед применением Lasso!
