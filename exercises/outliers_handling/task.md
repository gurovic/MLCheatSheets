# Упражнение: Обработка выбросов (Outliers Handling)

## Задание

Дан файл `input.csv` с данными, содержащими выбросы. Необходимо:

1. Для каждого признака (feature1, feature2, feature3) определить выбросы методом **IQR (Interquartile Range)**

2. Удалить строки, содержащие выбросы хотя бы в одном из признаков

3. Сохранить только колонку 'id' оставшихся строк

## Метод IQR для определения выбросов

1. Вычислить Q1 (25-й перцентиль) и Q3 (75-й перцентиль)
2. Вычислить IQR = Q3 - Q1
3. Определить границы:
   - Нижняя граница = Q1 - 1.5 × IQR
   - Верхняя граница = Q3 + 1.5 × IQR
4. Значения вне этих границ считаются выбросами

## Ожидаемый формат выходного файла

CSV файл с одной колонкой: id

Только те id, где все признаки находятся в допустимых пределах (без выбросов)
