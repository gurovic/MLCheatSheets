# Генерация PDF

Этот репозиторий содержит скрипт для автоматической генерации PDF-файла со всеми шпаргалками по машинному обучению.

## Готовый PDF-файл

Готовый PDF-файл уже доступен в репозитории: **[ML_Cheatsheets_Complete.pdf](ML_Cheatsheets_Complete.pdf)**

- **422 шпаргалки**
- **392 страницы**
- **~17 МБ**

## Самостоятельная генерация PDF

Если вы хотите сгенерировать PDF заново (например, после добавления новых шпаргалок):

### Требования

- Python 3.6+
- Google Chrome или Chromium
- Python-библиотеки: PyPDF2

### Установка зависимостей

```bash
pip install PyPDF2
```

### Генерация PDF

```bash
python3 generate_pdf.py
```

Скрипт выполнит следующие действия:

1. Создаст титульную страницу с информацией о коллекции
2. Конвертирует каждый HTML-файл из директории `cheatsheets/` в отдельный PDF
3. Объединит все PDF-файлы в один файл `ML_Cheatsheets_Complete.pdf`

### Процесс генерации

Генерация займет несколько минут, так как скрипт обрабатывает все 422 HTML-файла. Вы увидите прогресс в консоли:

```
Using Chrome: /usr/bin/google-chrome
Found 422 cheatsheet files

Creating cover page...
  ✓ Cover page created

Converting HTML files to PDF...
  [1/422] 1d_3d_cnn_cheatsheet.html... ✓
  [2/422] 3d_computer_vision_cheatsheet.html... ✓
  ...
```

### Примечания

- Скрипт использует headless Chrome/Chromium для конвертации HTML в PDF
- Временные PDF-файлы создаются во временной директории и автоматически удаляются после объединения
- Итоговый PDF оптимизирован для печати и чтения
